# Админка новостей

## Запуск

Из корня проекта:

```bash
npm run admin
```

Адреса после запуска:

- сайт: `http://127.0.0.1:43119/`;
- админка: `http://127.0.0.1:43119/admin/`.

Сервер доступен только на этом компьютере. Из локальной сети и интернета админка не открывается.

## Возможности

- поиск и фильтр по году;
- создание новости;
- редактирование заголовка, даты, адреса, описания и текста;
- загрузка основного изображения в JPG, PNG или WebP;
- удаление новости с подтверждением;
- автоматическая пересборка главной, архива, страницы новости и карты сайта;
- автоматическая проверка внутренних ссылок после каждого изменения.

Исходные данные находятся в `v3/data/news.json`. Перед изменением сервер создает резервную копию в `.admin-backups/`.

## Публикация

Локальная версия уже подходит для наполнения и проверки сайта. Перед передачей доступа менеджеру нужен закрытый сервер с авторизацией и автоматической публикацией собранных файлов на основной хостинг. Публично размещать текущий локальный API нельзя.
