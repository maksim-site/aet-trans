31.05.2019 г был доставлен груз «Парогенератор ПГВ-1000М», изготовленного филиалом АО «АЭМ-технологии» «Атоммаш» в г. Волгодонск для нужд филиала АО «Концерн Росэнергоатом» -Балаковской АЭС. Параметры груза: 14400х5700х5160 мм, вс брутто 311700 кг.

Транспортировка была выполнена по маршруту: г. Волгодонск, Ростовская область - Балаковская АЭС, Саратовская область.

Для реализации проекта потребовалось сооружение дополнительной инфраструктуры с привлечением сторонних организаций.

Партнерами по данному проекту выступили: ООО «ТРАК-СЕРВИС»и ООО «ВИКАР Логистик».
