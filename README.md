# Сайт АЕТ Транс

Статический сайт транспортно-экспедиторской компании с сохранением основных адресов старого WordPress-сайта.

## Что входит

- главная и основные корпоративные разделы;
- услуги, негабаритные перевозки и проекты;
- клиенты и отзывы с оригинальными материалами;
- архив из 191 новости за 2006-2026 годы;
- 552 локально сохраненные фотографии новостей;
- адаптивная версия и мобильное меню;
- интерактивная карта офиса и блок партнеров;
- локальная админка для добавления, редактирования и удаления новостей;
- `sitemap.xml`, `robots.txt`, canonical и Open Graph;
- старые адреса публикаций и служебные перенаправления.

## Локальный запуск

```bash
npm run admin
```

После запуска сайт доступен по адресу `http://127.0.0.1:43119/`, админка новостей по адресу `http://127.0.0.1:43119/admin/`.

На GitHub Pages по адресу `https://maksim-site.github.io/aet-trans/admin/` доступна демонстрация админки без пароля. В ней можно изменить существующую новость и открыть одноразовый предпросмотр на сайте. После обновления страницы исходная публикация восстанавливается, а изменения никуда не отправляются.

## Сборка

Основная версия:

```bash
npm run build
```

Резервная версия `/v3/`:

```bash
node scripts/generate-site.mjs v3
```

Проверка внутренних ссылок и файлов:

```bash
npm run validate
```

Подробности по работе с новостями: [`docs/ADMIN.md`](docs/ADMIN.md).

## Перед запуском

- согласовать и подключить Яндекс Метрику;
- утвердить политику обработки персональных данных;
- подтвердить актуальность логотипов клиентов и отзывов;
- выбрать закрытый способ размещения админки и настроить авторизацию;
- подключить хостинг, домен и HTTPS.
