# Сайт АЕТ Транс

Статический сайт транспортно-экспедиторской компании с сохранением основных адресов старого WordPress-сайта.

## Что входит

- главная и основные корпоративные разделы;
- услуги, негабаритные перевозки и проекты;
- клиенты и отзывы с оригинальными материалами;
- архив из 191 новости за 2006-2026 годы;
- 552 локально сохраненные фотографии новостей;
- адаптивная версия, мобильное меню и формы;
- `sitemap.xml`, `robots.txt`, canonical и Open Graph;
- старые адреса публикаций и служебные перенаправления.

## Локальный запуск

```bash
python3 -m http.server 8000
```

После запуска сайт доступен по адресу `http://127.0.0.1:8000/`.

## Сборка

Основная версия:

```bash
node scripts/generate-site.mjs
```

Резервная версия `/v3/`:

```bash
node scripts/generate-site.mjs v3
```

Проверка внутренних ссылок и файлов:

```bash
node scripts/validate-site.mjs
```

## Перед запуском

- выбрать канал отправки заявок, сейчас используется переход в почтовый клиент;
- согласовать и подключить Яндекс Метрику;
- утвердить политику обработки персональных данных;
- подтвердить актуальность логотипов клиентов и отзывов;
- выбрать способ публикации новых материалов;
- подключить хостинг, домен и HTTPS.
